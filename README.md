# 🚀 Job Application Tracker

## 📋 Overview
A collaborative job application tracking platform designed specifically for job seekers. Keep track of job applications, update statuses, and maintain a shared repository of potential opportunities with real-time updates between users.
✨ Features

🔐 Secure Authentication: Firebase-powered login system for authorized users
🔍 Job URL Tracking: Add and organize job listings with direct URL access
📊 Status Tracking: Visual indicators showing application status (applied/pending)
👥 Applicant Attribution: See who applied for which position
🔄 Real-time Updates: Instantly share updates when any user applies for a position
💬 Chat Interface: Built-in messaging system for collaboration on applications
📱 Responsive Design: Works seamlessly on both desktop and mobile devices
📈 Activity Feed: Track all actions and changes within the system

## Frontend:

React.js with Hooks for state management
React Router for navigation
Custom CSS with responsive design


## Backend:

Firebase Authentication for user management
Firebase Firestore for real-time database
Firebase hosting for deployment


## Additional Tools:

Modern ES6+ JavaScript
Component-based architecture
Custom theming with CSS variables

## 📱 Features Walkthrough
User Authentication
Secure login system using Firebase Authentication, ensuring that only authorized users can access the application.
Dashboard
The main interface with three primary tabs:

Jobs: View, add, and manage job applications
Chat: Communicate with other users about opportunities
Activity: Track recent actions and changes

## Job Management

Add new job listings with company, position, URL, and description
Update application status (Applied/Pending)
View detailed information about each opportunity

Real-time Collaboration
All changes sync instantly between users through Firebase Firestore, allowing for seamless collaboration.
